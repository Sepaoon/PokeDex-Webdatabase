🔑 API Keys — What's Needed & What's Optional

None of these keys are required to use PokéDexCheck. The app works out of the box with sensible free fallbacks. Adding keys unlocks better rate limits and extra features — each key below explains what it does, whether it's optional, and how it improves performance.

🃏 Pokémon TCG API Key Recommended ↗ dev.pokemontcg.io

Powers virtually every card-data feature in the app: the name/set autocomplete, card validation badges, card art on every tile, price lookups, set-info for the Setup Wizard, and the Master Set pre-fill.

Without key 1,000 requests/day · 30 per minute. Bulk price updates and large collections will hit this limit.
With key 20,000 requests/day — 20× more. Free signup, no credit card.
🤖 Google Gemini API Key Recommended for scanning ↗ Google AI Studio

Powers the 📷 camera card scan — reads the card's name, set, and number from a photo. Gemini gives the highest accuracy of the available options.

Without key Scanning falls back to OCR.Space if that key is set, otherwise to manual entry.
With key Generous free tier (check current limits at Google AI Studio). Zero setup beyond pasting the key.
🔤 OCR.Space API Key Optional alternative ↗ ocr.space

An alternative OCR provider for card scanning. Useful if you don't want to use Google. Less accurate than Gemini for card images but still workable.

Fallback order Gemini (if key) → OCR.Space (if key) → manual entry.
Free tier 500 requests/day. No credit card required.

Provides a live USD ↔ EUR rate for the Cardmarket price source and total-value display.

Default App uses the Frankfurter API (free, no key, ECB data). Accurate and updates daily — most users never need to add an OXR key.
With key Uses Open Exchange Rates as primary (hourly updates). Free tier: 1,000 requests/month.

📏 Quick Condition Guide

Look at the card under bright light. Check the edges, corners, surface, and centering. Pick the best match below — when in doubt, drop a grade lower (collectors prefer honest assessments).

M
Mint — Looks identical to a card just pulled from a freshly-opened pack. Sharp corners, no whitening, surface mirror-clean. Almost no card meets this in real-world use.
NM
Near Mint — Looks "new" at arm's length. Minor flaws only — maybe a tiny edge whitening visible only under close inspection. Default for cards from sleeved/binder storage. Most uncirculated cards land here.
LP
Lightly Played — Visible minor wear: light edge whitening, very faint surface scratches, small corner softness. No creases. Still tournament-legal.
MP
Moderately Played — Clear wear: noticeable corner whitening, light surface scratches you can spot without squinting, maybe a small bend or print line. The card is still fully readable and intact.
HP
Heavily Played — Heavy wear: significant edge/corner damage, visible creases, heavier scratches or scuffs, but the card is intact and identifiable.
DMG
Damaged — Major issues: tears, water damage, severe creasing, missing chunks, heat-warping, ink stains, or anything that affects the structure of the card.
💡 Quick test: Hold the card by the edges, tilt under good light. Whitening shows up as lighter dots/lines on dark borders. Scratches catch the light. Bends are visible from the side. Centering is the biggest factor for grading-grade cards — the borders should be even on all four sides.

What kind of binder do you want to build?

Pokédex / Master Set / Deck collect what you have. Chase / Trade lists track what you want, or what you'll trade away.

📘 Pokédex One slot per Pokémon species (National or regional dex).
🃏 Master Set One slot per card — complete a specific set or every card of a single Pokémon.
🎴 Deck A 60-card playable deck. Tracks Pokémon / Trainer / Energy ratios and format legality.
🎯 Chase List Cards you want to acquire. Each entry's value is your max budget for that card.
🤝 Trade List Cards you'll trade or sell. Each entry's value is your asking price.
